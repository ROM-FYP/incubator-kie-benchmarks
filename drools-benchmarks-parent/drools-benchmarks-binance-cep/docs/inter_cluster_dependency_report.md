- **Fallback ŌåÉ Cluster 2:** via `[BestBidAsk, DepthState, SpreadState, VolState]`
- **Fallback ŌåÉ Cluster 3:** via `[SignificantTrade]`
- **Fallback ŌåÉ Cluster 4:** via `[LiquidationStats]`
- **Fallback ŌåÉ Cluster 5:** via `[TradeStats]`
- **Cluster 1 ŌåÉ Fallback:** via `[FeedHealth, ModeState]`

**Conclusion on Fallback:** The Fallback heavily depends on the outputs of **all 5 clusters**. This mathematically verifies our previous empirical finding: if the Fallback is isolated without a shared working memory with the clusters, it will suffer catastrophic data loss (e.g., 99.2% RiskSignal drop) because its rules will be starved of the intermediate state generated by the parallel clusters.
