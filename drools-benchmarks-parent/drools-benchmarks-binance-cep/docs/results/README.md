# Benchmark Results

This directory will contain benchmark results including:

- CSV files with JMH benchmark results
- Flamegraph HTML files
- JFR recording files
- Performance analysis reports

## File Naming Convention

### JMH Results
- `binance_cep_baseline_YYYYMMDD_HHMMSS.csv`
- `binance_throughput_YYYYMMDD_HHMMSS.csv`
- `binance_scalability_YYYYMMDD_HHMMSS.csv`

### Flamegraphs
- `flamegraph_cpu_YYYYMMDD_HHMMSS.html`
- `flamegraph_alloc_YYYYMMDD_HHMMSS.html`

### JFR Files
- `jfr_recording_YYYYMMDD_HHMMSS.jfr`

## Analysis Reports
- Performance analysis documents
- Comparison reports
- Optimization recommendations
