\section{Conclusion and Future Work}\label{sec:conclusion}

We presented a multi-domain CEP benchmark suite covering aviation, financial trading, and collaborative editing, each pairing a domain-derived rule set with ${\sim}1.6$M real-world events and a reproducible replay harness built on \texttt{incubator-kie-benchmarks}. The suite's three domains naturally exhibit different workload profiles --- join-heavy, chain-deep, and pipeline-sequential --- with high baseline rule coverage ($>$63\%) validating that real-world streams effectively exercise the rule sets. Coverage-variant analysis further shows that activation depth is governed by stream diversity rather than event volume.

Future work includes scaling the rule bases to larger rule counts to study how engine performance degrades with increasing rule-base complexity, and conducting cross-engine comparisons using these benchmarks to evaluate how different production rule engines handle identical CEP workloads.
